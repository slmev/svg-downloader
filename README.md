# SVG 批量下载助手 (SVG Bulk Downloader Assistant)

SVG 批量下载助手是一款功能强大的浏览器扩展，旨在帮助设计师和开发者一键提取并下载当前网页中的所有 SVG 图片。

## 🌟 主要功能

- **智能扫描**：自动识别网页中的内联 SVG 标签和外部引用的 SVG 文件。
- **实时预览**：提供所有发现的 SVG 图片列表及预览图，方便快速识别。
- **批量下载**：支持多选、全选及反选，一键将选中的 SVG 打包为 ZIP 压缩包下载。
- **搜索过滤**：内置搜索功能，可根据文件名快速筛选所需的图标。
- **支持中文**：完美支持中文文件名和界面显示。
- **进度显示**：实时展示打包下载进度，操作状态一目了然。

## 🚀 安装说明

由于本项目目前为开发版本，您可以按照以下步骤手动加载到浏览器中：

1. 下载或克隆本项目代码到本地。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/` 页面。
3. 开启页面右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”** 按钮。
5. 选择本项目所在的文件夹即可完成安装。

## 🛠️ 使用方法

1. 在任何包含 SVG 的网页上，点击浏览器工具栏中的扩展图标。
2. 扩展会自动开始扫描当前页面的所有 SVG 图片。
3. 点击 **“预览列表”** 展开详情，您可以：
   - 使用搜索框查找特定的 SVG。
   - 勾选或取消勾选特定的图标。
   - 使用 **“全选”** 或 **“反选”** 快速操作。
4. 点击 **“批量下载”** 按钮，系统将自动打包并下载选中的文件。

## 📁 文件结构

- `manifest.json`: 扩展配置文件（Manifest V3）。
- `content.js`: 页面内容脚本，负责提取 SVG 数据。
- `popup.html` & `popup.js`: 扩展弹出界面的结构与逻辑。
- `jszip.min.js`: 用于生成 ZIP 压缩包的第三方库。
- `icon.png`: 扩展图标。
